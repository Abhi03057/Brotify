🎵 Brotify
Brotify is a sleek, Spotify-inspired music player built using HTML, CSS, and JavaScript. Designed for music lovers, Brotify offers an engaging user interface and smooth playback experience — all in the browser.

🚀 Features
🎧 Play/pause your favorite tracks

⏭ Skip to next or previous song

🔁 Loop and shuffle modes

🎚️ Real-time progress bar and seek control

📱 Responsive design for both desktop and mobile

✨ Smooth animations and transitions

🛠️ Tech Stack
  HTML5 – Markup structure
  
  CSS3 – Styling and responsiveness
  
  JavaScript (Vanilla) – Music controls and interactivity
